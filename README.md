# Foil library

A library supposed to be zero overhead in regard to syscall utilities, and to have decent primitives. The focus of it is to deliver code that's as efficient as possible, and to make the development of performance sensitive software simpler.

It's being designed to allow the use of only this library, thus kernel constants regarding the syscalls are in the lib.

This library has no intention on shipping code to any OS besides linux, therefore any pull request intended to create compatibility to others are not going to be merged.
