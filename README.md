# Foil library

A library supposed to be zero overhead in regard to syscall utilities, and to have decent primitives. The focus of it is to either deliver code that's extremelly specialized and fast, to make easier the development of performance sensitive software.

It's being designed in a way to allow you to use only this library, thus kernel constants regarding the syscalls are in the lib.

This library has no intention on shipping code to any OS besides linux, therefore any pull request intended to create compatibility to others are not going to be merged.
